Install SchemaQualityChecker as an Eclipse/WSAD Plugin

SchemaQualityChecker can be installed as an Eclipse or WSAD plugin. Here are the installation steps to follow:

  1. Shut down Eclipse or WSAD;
  2. Create a new directory under the EclipseDirOrWSADDir/plugins directory (EclipseDirOrWSADDir refers to the directory where Eclipse or WSAD has been installed) called com.ibm.etools.sketch.sqc (If a previous version of SQC Plugin was already installed, first remove the existing EclipseDirOrWSADDir/plugins/com.ibm.etools.sketch.sqc directory);
  3. Unzip SQC2.2.zip and put its content under the newly created directory;
  4. Restart Eclipse or WSAD.

Please see Using SQC as an Eclipse/WSAD Plugin for more details on how to invoke SQC from Eclipse or WSAD.
