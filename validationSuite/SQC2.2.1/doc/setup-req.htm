Requirements

SchemaQualityChecker (SQC) and QualityChecker (QC) API require a Java Runtime Environment (JRE) version 1.3 or greater.

IBM's version of the JRE (1.3) can be downloaded after registering at WebSphere Preview Technologies and Sun's version can be obtained at http://java.sun.com/j2se/1.4/jre/. IBM's JRE comes with a xerces.jar and xml4j.jar files included. These files should be moved to another directory.

The xmlParserAPIs.jar, xercesImpl.jar and xml4j.jar files included in the Schema Quality Checker distribution package should be used to avoid run-time errors.

Users who would like to use Microsoft's Internet Explorer to see the XML report generated by SchemaQualityChecker with the stylesheet provided in the distribution package, should use either version 5.5 and MSXML 3.0 (installed in replace mode) or version 6.0.
